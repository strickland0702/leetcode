# 位运算

#### [剑指 Offer 15. 二进制中1的个数](https://leetcode-cn.com/problems/er-jin-zhi-zhong-1de-ge-shu-lcof/)

```python
class Solution:
    def hammingWeight(self, n: int) -> int:

        count = 0

        while n != 0:
            # n & n-1 可以将最右侧的1变为0
            n = n & (n-1)
            count += 1

        return count
```
